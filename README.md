## 📌 Descripción

Este repositorio presenta una **infografía explicativa de Docker Engine**, diseñada para entender de manera visual y sencilla cómo funciona la arquitectura interna de Docker.

Está orientado a estudiantes, desarrolladores y cualquier persona que quiera iniciarse en **contenedores y DevOps**.

---

## 🧠 ¿Qué aprenderás?

- Qué es **Docker Engine**
- Cómo funciona la arquitectura cliente-servidor
- Interacción entre:
  - Docker CLI
  - REST API
  - Docker Daemon
- Gestión de:
  - Contenedores
  - Imágenes
  - Redes
  - Volúmenes

---

## 🏗️ Arquitectura de Docker

Docker Engine se compone de tres elementos principales:

### 🔹 Docker Daemon (Servidor)
Servicio que corre en segundo plano y administra:
- Contenedores
- Imágenes
- Redes
- Volúmenes

---

### 🔹 REST API
Permite la comunicación entre cliente y servidor.

---

### 🔹 Docker CLI (Cliente)
Interfaz de línea de comandos usada para ejecutar instrucciones como:
